Developer Docs
==============

Requirements
------------

* apk-factory-service
* Java
* Ant
* Android SDK

Running the service - See docs in apk-factory-service repo

Libarary Version
----------------

The file `VER.txt` contains a single integer that is the current version number.
This will get loaded by the service on startup.
